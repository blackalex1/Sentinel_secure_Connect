# Sentinel

**Sentinel** — это Android-клиент для защиты сетевого трафика и анализа сетевой активности на базе встроенного ядра **Xray-core** (VLESS, VMess, Trojan, Shadowsocks).

## Описание проекта
Приложение обеспечивает шифрование и защиту трафика на уровне ядра ОС Android с помощью TUN-интерфейса, поддерживает раздельное туннелирование приложений (Split Tunneling), фильтрацию маршрутов по географическим правилам (GeoIP/GeoSite) и аудит подозрительных сетевых соединений в целях повышения конфиденциальности пользователей.

## Требования для сборки
* **JDK:** 21
* **Android SDK compileSdk:** 36 (targetSdk: 28, minSdk: 28)
* **Gradle:** Версия, сконфигурированная в проекте.

## Сборка
Для сборки отладочной версии выполните команду:
```bash
./gradlew assembleFossDebug
```
